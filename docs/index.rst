lww-transport
=============

.. figure:: _static/wigner_function.png
   :alt: Steady-state Wigner function
   :width: 80%
   :align: center

   Wigner distribution function for a symmetric GaAs-AlGaAs-GaAs RTD with 30 Angstrom
   barrier and 50 Angstrom quantum-well widths with zero bias at T = 70K

``lww-transport`` is a reusable Python package for one-dimensional
Lattice Weyl-Wigner / Wigner-Poisson quantum transport simulations.

The package exposes a configurable simulator, legacy CSV I/O, and a
command-line interface for steady-state and transient simulations.
